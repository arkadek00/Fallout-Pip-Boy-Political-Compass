# ☢️ Fallout Pip-Boy Political Compass

Interactive Fallout-style political compass with factions, ideology mapping and immersive Pip-Boy UI.

---

## 🚀 Getting Started

No installation required.
Just click on the project link:
👉 https://arkadek00.github.io/Fallout-Pip-Boy-Political-Compass/

---

## 🧠 Overview

This project is an interactive visualization inspired by the Fallout universe, combining a political compass system with in-world factions.
It allows users to explore ideological positioning, compare factions, and interact with a stylized Pip-Boy interface.

---

## ✨ Features

* 🧭 56 Fallout factions mapped on a political compass
* 🧬 Ideology DNA system – multi-dimensional faction traits
* 🕰️ Timeline system – explore factions across different eras
* 🎮 Game filtering – filter factions by Fallout titles
* 🏷️ Faction type system – categorize by structure and ideology
* 🎛️ Layer system – toggle different data layers
* ⚡ Presets – quick view configurations
* 🔍 Compare mode (Shift + Click) – analyze differences between factions
* 🎧 Ambient Vault-style audio (Web Audio API)
* 💡 Fully interactive, responsive UI inspired by Pip-Boy aesthetics

---

## 🎮 Controls

| Action           | Result                    |
| ---------------- | ------------------------- |
| Click on faction | Focus on selected faction |
| Shift + Click    | Compare multiple factions |
